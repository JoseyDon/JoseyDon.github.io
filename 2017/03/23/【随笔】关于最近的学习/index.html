【随笔】关于最近的学习

​ 最近参加了IFE2017的前端学习，我是在2016年末了解到百度前端学院的。由于HTML+css部分在过去已经完成了，并且做了一些切图练习，这段时间就主要在学习JavaScript。

​ 我第一次接触JavaScript是因为有位学长说，“做个轮播图吧！”。当时一直在练习静态页面的我两眼茫然，在网上参考了很多很多的代码，看不懂的代码挨着在网上搜索，还在慕课网上看了教程，学了慕课上JavaScript的入门篇，但最终还是实现不了。不过话说回来，在学习dom的时候，真的很有意思，但当时的我为了实现轮播功能急的焦头烂额，根本没工夫去感叹。

​ 之后学习了《JavaScript DOM编程艺术》，终于有点入门的感觉。在网上参考代码也搞得懂了，基本上只要看懂别人的逻辑自己打就没问题了。这也是我前段时间完成IFE作业的过程。在参考别人的代码时，我收获了很多。不过，还是觉得不满意啊。总觉得自己还是没有逻辑，没有思想，这也是我现阶段需要去获得了技能。

​ 写这篇随笔的目的，只是想表达学习代码真的好有意思，那种把零乱的效果实现的过程，看别人的学习路径，用键盘在屏幕上敲出一个一个符号，我们把这些符号的不同排列赋予了不同的意义，组合起来带给我们视觉冲击。学习编程真是太有意思了。

​ 写于2017.3.23年上午 一个停课的早上
